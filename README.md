# A programming language interpreter written in go

## Development
### Run test
```shell script
go test ./...
```

### Repl
```shell script
go run main.go
```

## Getting started

### Variable
```swift
>> let x = 10;
>> x
10
>> let y = 5;
>> y
5
```

### Opeartor
```swift
>> let x = 5;
>> let y = 10;
>> x + y;
15
>> 2 + (3 + 4) * 5 + 6;
43
>> 5 * 5 + 10
35
>> 3 + 4 * 5 == 1 + 4 * 5
false
```
